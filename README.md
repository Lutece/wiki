## 작성규칙
  - 문서 생성은 GFM (Github Flavored Markdown) 을 사용한다. (확장자 .md)
  - 언어나 기술명 등으로 폴더를 만든다.
  - 파일명은 영어로.
  - README.md에 링크 추가하기.
  - 절대 어뷰징은 없다.
  - 1일 1커밋을 원칙으로 하되 너무 강제성을 부여하지 않는다.
  - 커밋메세지에 신경 쓸 것.
